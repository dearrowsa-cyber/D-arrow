## 🔤 Typography System

### Font Families

#### Gilroy (Headlines & Display)
- **Weight:** Bold (800), Semi-Bold (700), Medium (600)
- **Usage:** H1, H2, H3, Display text, Feature titles
- **Characteristics:** Modern, geometric, confident
- **Import:** `https://fonts.googleapis.com/css2?family=Gilroy:wght@400;500;600;700;800&display=swap`

#### TT Hoves Pro (Body & UI)
- **Weights:** Regular (400), Medium (500), Semi-Bold (600), Bold (700)
- **Usage:** Body text, UI elements, buttons, form labels
- **Characteristics:** Clean, professional, highly readable
- **Note:** Web-safe fallback to system fonts

#### 29LT Bukra (Arabic Content)
- **Usage:** All Arabic language content
- **Weights:** Regular (400), Semi-Bold (600), Bold (700)
- **Characteristics:** Modern Arabic sans-serif

### Typography Hierarchy

#### H1 - Page Titles
- **Font:** Gilroy 800
- **Size:** Responsive clamp(48px, 8vw, 72px)
- **Line Height:** 1.1
- **Letter Spacing:** -0.02em
- **Color:** `#1F2937`

#### H2 - Section Headers
- **Font:** Gilroy 700
- **Size:** Responsive clamp(32px, 6vw, 48px)
- **Line Height:** 1.2
- **Color:** `#1F2937`

#### H3 - Subsection Headers
- **Font:** Gilroy 600
- **Size:** 24px
- **Line Height:** 1.2
- **Color:** `#1F2937`

#### Body Text
- **Font:** TT Hoves Pro 400
- **Size:** 14-16px
- **Line Height:** 1.6
- **Color:** `#6B7280`

#### Button Text
- **Font:** TT Hoves Pro 600
- **Size:** 14-16px
- **Line Height:** 1.4
- **Letter Spacing:** 0.02em

### Language Support
- **Primary:** English (US/UK)

### RTL (Right-to-Left) Guidelines
- Flexbox: `flex-row-reverse` for Arabic layouts

## 📱 Responsive Design

### Breakpoints
- **Mobile:** 320px - 639px
- **Tablet:** 640px - 1023px
- **Desktop:** 1024px+
- **Large Desktop:** 1280px+

### Typography Scaling
- Headings use responsive `clamp()` functions
- Base font size: 16px (mobile) to 18px (desktop)
- Line heights maintain minimum 1.5 for body text

## ✅ Brand Usage Guidelines

### DO ✅
- Use official brand colors in gradients for maximum impact
- Maintain minimum contrast ratios (WCAG AA)
- Use Gilroy for all headlines and display text
- Use TT Hoves Pro for body content and UI
- Keep pink as primary action color
- Use subtle brand gradients as accents

### DON'T ❌
- Don't use other colors in the primary palette
- Don't distort or rotate the logo excessively
- Don't use unauthorized font variations
- Don't place logo on busy backgrounds without contrast
- Don't reduce logo below 32px
- Don't use outdated color combinations

## 📊 Accessibility

### Color Contrast
- **Text on Pink:** White text on `#FF4D6D` = 4.5:1 ratio (WCAG AA)
- **Text on Orange:** White text on `#FF9A3C` = 4.5:1 ratio (WCAG AA)
- **Text on Light BG:** Dark text `#1F2937` on `#F9FAFB` = 15:1 ratio (AAA)

### Font Sizing
- Minimum 12px for body text
- Minimum 16px for mobile device text
- Responsive scaling for readability

### Focus States
- All interactive elements have visible focus states
- Focus color: Brand pink `#FF4D6D`
- Minimum focus outline: 2-3px

---

## 📈 Performance Considerations

### Font Optimization
- Google Fonts API with controlled weights
- Fallback to system fonts (TT Hoves Pro)
- Font-display: swap for non-blocking loading

### Color Optimization
- Use CSS custom properties for consistent theming
- Gradient backgrounds use hardware acceleration
- Minimal use of filters for performance
